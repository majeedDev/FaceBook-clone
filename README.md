# Facebook-clone Project

## Overview

This is ruby on rails project with two table:
* Users table (name, E-mail, age, gender)
* Posts table (title, description, user_id)

## Technologies Used

* Ruby 3.0.3

* Ruby on Rails (RoR) 7.1

* Sqlite3 DataBase

* Visual Studio (VS Code)
